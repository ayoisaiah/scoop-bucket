# scoop-bucket

Scoop bucket for all my open source projects, powered by [goreleaser](https://github.com/goreleaser/goreleaser).

## Install F2

```bash
$ scoop bucket add ayoisaiah-scoop-bucket https://github.com/ayoisaiah/scoop-bucket
$ scoop install f2
```

## Install Focus

```bash
$ scoop bucket add ayoisaiah-scoop-bucket https://github.com/ayoisaiah/scoop-bucket
$ scoop install focus
```
